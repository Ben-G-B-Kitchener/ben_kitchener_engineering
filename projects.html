Projects

ELSA Cryogenic Facility

A cryogenic test system for investigating superconductors and their jointing technologies, involving large currents and magnetic fields at cryogenic temperatures down to 20 K (-253 °C). Responsible for SAT delivery of the C&I system at UKAEA South Yorkshire.

ATLAS ITk Upgrade

Department of Physics & Astronomy, The University of Sheffield

Supporting the ATLAS ITk upgrade programme at CERN. Work included cleanroom assembly, wire bonding, TIG welding of titanium cooling tubes, instrumentation design, PCB layout, and firmware development.

Side project: Remote Environment Monitoring System (REMS) using an 868 MHz wireless sensor network.

Turbine Blade Remanufacturing System

Systems integration demonstrator combining robotic welding, scanning systems, and control software using LabVIEW cRIO for an Innovate UK project.

Single Manufacturing Platform Environment (SIMPLE)

Real-time TIG arc monitoring system using NI PXI + LabVIEW. Classified weld process faults and enabled automated safe shutdown logic.
